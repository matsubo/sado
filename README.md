# Sado triathlon type A athlete tracker

## backend

reuqirements: Docker, docker-compose, python3

Scrape original list and upload combined CSV file to the gist.
```
cd backend
bash loop.sh
```

## frontend


reuqirements: NodeJS

```
cd front
npm start
open 'http://localhost:3000/'
```

## Deployment

This project is ready for Vercel.
